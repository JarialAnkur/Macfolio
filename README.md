# Macfolio
Macfolio is a Project, initiated as a Personal Portfolio website but It is much more than that.

# How to use it
* Clone the project 
* Get to project directory
* To start the app use
```
"start": "node server.js",
```
* Run app in dev mode
```
"dev": "nodemon server.js"
```
